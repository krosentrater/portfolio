Objective:

This project is intended to utilize Django and Python to create an E-commerce website where users can login or create an account. Once logged in they have opened access to the rest of the features for discounts, shopping, etc.

What I Learned:

As this became a bigger and bigger application, I really learned the value of namespacing and organization to help me keep everything in order. I learned that Django has many useful built in features and functions. As this is still a work in progress, I plan to keep adding to it to continue to grow my knowledge. My plans include a better database model for Django, an add to cart feature, and a check out feature.
