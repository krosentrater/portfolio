Objective:

The project is meant to establish a connection with the MySQL database and a React application. Creates a type of "to-do" list where you can add recycled items, edit those items, or delete them.

What I Learned:

Establishing a connection with MySQL was very similar to MongoDB but with different libraries. The queries can be slightly difficults when ensuring variables using template literals are working properly. Helped me gain an understanding of how to use MySQL as a database for a front-end application.
