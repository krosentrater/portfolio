Objective:

The project is meant to establish a connection with the MongoDB database and a React application. Demonstrates the successfull communication of queries, creating an account, logging into an acccount, posting comments, and establishing user authentication with password hashing.

What I Learned:

Establishing a connection with MongoDB through React was a learning process. Creating the models and queries from the front-end was a delicate process. Ensuring that the password was hashed at a basic level was complicated, but dedication I was able to progress through and gain a real understanding of how many current social media sites work.
